# Vizyonda_Sinema
Bu proje, vizyondaki ve yakında gösterime girecek filmlerin listelendiği kapsamlı bir sinema web sitesidir. Kullanıcılar film detaylarını inceleyebilir, vizyondaki filmler için bilet satın alabilir. Bilet alımı için giriş zorunludur. Şehir, tarih ve seans seçimine göre en uygun indirim uygulanır. Sayfalar arası aktif menü mevcuttur.
